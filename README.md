# INST377FinalProject
My final project for INST377, named F1 STATISTICS.
F1 STATISTICS is a fun, cool looking website that serves as a more direct competitor to the constantly bloated official Formula 1 website.
Target browsers include Edge and Chrome, not sure how the website would interact or appear on iOS or Android.

# Developer Manual
If you would like to run this application on your machine and be able to make the proper edits to the website, make sure that you first npm install supabase, body-parser, dotenv, express, nodemon, and the usa-state-validator. The main bug I am currently faced with is still making sure that the table self-deletes when you search for more than one driver/team without the need to refresh.
